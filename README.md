# 🐶 宠物选课系统

## 📌 项目介绍
本项目是一个基于前后端分离架构开发的宠物选课系统，主要面向宠物主人，提供宠物训练课程管理与学习成长功能。

系统借鉴“学生选课系统”的设计思路，以“宠物”为学习主体，实现宠物课程选择、训练任务、学习统计、技能成长等功能，提升宠物训练的管理效率与趣味性。

---

## 🚀 技术栈
- 前端: Vue + Axios + Element UI + ECharts
- 后端: SpringBoot + MyBatis
- 数据库: MySQL

---

## 🧩 功能模块
- 📚 课程列表：查看所有宠物训练课程
- 🐾 我的宠物：管理宠物信息
- 📖 宠物课程：查看宠物已选课程
- 🏆 排行榜：根据宠物经验值进行排名
- 🎯 训练任务：完成课程任务获取经验
- 📈 宠物成长：记录宠物等级与经验变化
- 🎖 宠物勋章：完成任务获得成就奖励
- 📊 学习统计：通过图表展示学习情况（ECharts）
- 🌳 宠物技能树：展示宠物技能成长路径
- 🤖 课程推荐：根据宠物类型推荐课程

---

## 🌟 项目亮点
- ✅ 前后端分离架构（Vue + SpringBoot）
- ✅ 数据可视化（ECharts 学习统计）
- ✅ 游戏化设计（排行榜、勋章、技能树）
- ✅ 宠物选课模式（创新点）
- ✅ 简单推荐算法（课程推荐功能）
