# 宠物成长课程平台

## 项目介绍
基于 Vue + SpringBoot + MySQL 的宠物选课系统，实现宠物课程管理与学习成长全流程功能，为宠物主人提供课程选择、成长记录、数据统计一体化服务。

## 功能模块
- 课程列表：展示全品类宠物训练/养护课程，支持筛选/搜索
- 我的宠物：宠物信息管理、成长档案录入
- 宠物课程：已报名课程管理、学习进度追踪
- 排行榜：宠物学习成果/打卡活跃度排名
- 训练任务：课程配套任务打卡、完成度统计
- 宠物成长：成长数据记录、趋势可视化
- 宠物勋章：学习成就勋章解锁、游戏化激励
- 学习统计：基于 ECharts 实现学习时长/完成率可视化
- 宠物技能树：技能解锁进度、能力成长路径
- 课程推荐：基于宠物品种/年龄的智能课程推荐

## 技术栈
- 前端：Vue 3、Axios、Element UI、ECharts
- 后端：SpringBoot 2.x、MyBatis
- 数据库：MySQL 8.0

## 项目亮点
- 前后端分离架构：解耦前端与后端，便于维护扩展
- 数据可视化：基于 ECharts 实现学习数据多维度统计
- 游戏化设计：勋章、技能树、排行榜提升用户粘性
- 轻量化推荐系统：基于宠物属性实现个性化课程推荐
