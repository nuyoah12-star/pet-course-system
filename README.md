# 宠物成长课程平台

## 项目介绍
基于 Vue + SpringBoot + MySQL 的宠物选课系统，实现宠物课程管理与学习成长功能。

## 功能模块
- 课程列表
- 我的宠物
- 宠物课程
- 排行榜
- 训练任务
- 宠物成长
- 宠物勋章
- 学习统计（ECharts）
- 宠物技能树
- 课程推荐

## 技术栈
前端：Vue、Axios、Element UI、ECharts  
后端：SpringBoot、MyBatis  
数据库：MySQL  

## 项目亮点
- 前后端分离架构
- 数据可视化（学习统计）
- 游戏化设计（勋章、技能树、排行榜）
- 简单推荐系统（课程推荐）

## 启动方式
前端：npm run dev
后端：运行 SpringBoot 项目
